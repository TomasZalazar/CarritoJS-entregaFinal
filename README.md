## 👨🏽‍💻 Instalación 👨🏽‍💻

1. Clona este repositorio: `git clone https://github.com/TomasZalazar/CarritoJS-entregaFinal.git`
2. Accede a la carpeta del proyecto: `cd CarritoJS-entregaFinal`
3. Instala las dependencias: `npm install`
